Detailed Description

TinyBuffer represents object to work with double buffered display. Easy to use: TinyBuffer lcd( SSD1306_128x64 ); void setup() { lcd.beginI2C();

lcd.clear(); lcd.charF6x8(0,0,"Hello"); }
