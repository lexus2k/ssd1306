void ssd1306_i2cInit_Embedded ( uint8_t  scl,
uint8_t  sda,
uint8_t  sa 
)

Initializes software implementation of i2c. If you do not know i2c parameters, try ssd1306_i2cInit_Embedded(0,0,0).

Warning
the function disables interrupts.
Parameters
scl- i2c clock pin. Use 0 if you don't need to change default pin number
sda- i2c data pin. Use 0 if you don't need to change default pin number
sa- i2c address of lcd display. Use 0 to leave default
